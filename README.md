# Polynem Articles — Digital Sovereignty Essays & Analysis

A Next.js editorial publishing site for Richard St-Pierre's essays, articles, and policy reports on digital sovereignty, AI governance, and national strategy.

## Tech Stack

- **Next.js 15** (App Router)
- **Once UI Core** (design system + components)
- **MDX** (content authoring via `@next/mdx` + `next-mdx-remote`)
- **Vercel** (hosting)
- **Pandoc** (Word → Markdown conversion pipeline)

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Adding New Articles

### From a Word document:

```bash
# 1. Convert .docx to markdown
./scripts/convert-docx.sh path/to/article.docx "Article Title" "category"

# 2. Edit the generated MDX file in content/articles/ to verify frontmatter
# 3. Commit and push — Vercel deploys automatically
```

### Manually:

Create a new `.mdx` file in `content/articles/` with this frontmatter:

```mdx
---
title: "Your Article Title"
date: "2026-03-15"
category: "Policy"
author: "Richard St-Pierre"
publication: "Policy Magazine"
summary: "A one-sentence summary for the listing page."
readTime: "12 min read"
featured: false
---

Your article content in Markdown here.
```

## Deployment

Push to `main` branch → Vercel auto-deploys.

## Content Pipeline

Word (.docx) → Pandoc → Markdown → Add frontmatter → .mdx → Git push → Vercel deploy
